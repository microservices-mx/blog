FUNDAMENTOS GIT

Un taller diseñado para aquellas personas que tienen poca o nula experiencia con Git, mediante ejercicios prácticos descubre los beneficios de esta herramienta estándar de la industria y más popular en la web.

COLABORACIÓN CON GIT

Descubriras cómo Git puede mejorar drasticamente tu flujo de trabajo y ayudar desde la creación y publicación de código fuente y hasta sitios estáticos (con hospedaje de sitio 100% gratis!).

GITHUB Y GITLAB

Un taller con técnicas intermedias y avanzadas en Git donde además descubrirás el uso apropiado, las características y los beneficios de dos opciones muy populares de Git en la nube: GitHub y GitLab.

QUÉ ES DOCKER, POR QUÉ Y PARA QUÉ ME SIRVE

En este taller descubrirás el apasionante mundo de los contenedores Docker, desde los conceptos básicos, instalación y configuración, hasta la creación de aplicaciones basadas en contenedores.

CONFIGURACIÓN AVANZADA DE CONTENEDORES DOCKER

Conceptos y técnicas de configuración avanzada para Docker, descubrirás cómo se comunican los contenedores en una red, cómo crear redes sobre-puestas (overlay), configurar DNS y balanceadores de carga, incluyendo creación de clusters de almacenamiento en Docker.

ORQUESTACIÓN DE CONTENEDORES DOCKER

Revisaremos qué es la orquestación de contenedores y cómo adminstrar contenedores Docker a escala. Con ejercicios prácticos, revisaremos una aplicación de ejemplo que administraremos utilizando la orquestación.

FUNDAMENTOS EN DEVOPS Y MICROSERVICIOS

Las técnicas de desarrollo ágil de software, la migración de aplicaciones a la nube, la cultura DevOps, la integración continua y el despliegue continuo (CI / CD) y el uso de contenedores se utilizan junto con los microservicios para revolucionar el desarrollo y la entrega de aplicaciones. Descubre cómo.

MIGRACIÓN DE MONOLÍTOCO A MICROSERVICIOS

¿Tienes una solución monlítica que te gustaría mejorar? Con microservicios puedes, desarrolla o refactoriza tu aplicación en servicios separados que se "hablen" entre sí de manera bien definida. Cada microservicio es autónomo, mantiene sus datos, y cada uno puede actualizarse independientemente de otros. Descubre cómo, en este taller práctico.

SERVICE MESH

En la práctica, las arquitecturas basadas en microservicios presentan varios problemas para gestionar, configurar o de infraestructura, Service Mesh es una arquitectura que se ha vuelto muy popular porque resuelve esos problemas, aislando las aplicaciones de la infraestructura. En el taller descubrirás como aplicar actualizaciones en vivo sin afectar el entorno productivo.
